# Agents Overview

This project contains a variety of specialized AI agents, each designed for a specific purpose. Below is an overview of the available agents.

| Agent | Purpose | Key Feature |
| :--- | :--- | :--- |
| **Self-Evolving Agent** | Tool-creating agent with GUI | AST analysis & sandbox execution |
| **Book Writer** | Chapter-by-chapter book generation | GitHub auto-commit & resume capability |
| **Course Generator** | Multilingual course creation | 4-agent sequential pipeline |
| **Caveman Compressor** | Text compression | Terse technical summaries |
| **Tutorial Debug Agent** | ADK tutorials & debugging | Terminal error analysis |
| **Tiny Jarvis** | Telegram scheduling agent | Multi-agent team coordination |
| **Sci Paper Writer** | Scientific paper generation | 5-agent pipeline & source ingestion |
| **Tiny Company** | TOPIK learning assistant web app | End-to-end product development |

For detailed information on each agent, please visit their respective pages in the navigation menu.
