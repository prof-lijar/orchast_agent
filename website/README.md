# Orchast Agent Website

This directory is reserved for the Orchast Agent documentation landing page.

The dev-team agent should build the GitHub Pages website here. Keep the website
self-contained in this directory unless shared repository assets are reused from
`../assets`.

## Deployment target

Deploy with GitHub Pages from the `website/` project output.

Recommended implementation direction:

1. Use a static-site-friendly stack.
2. Keep build output compatible with GitHub Pages.
3. Document local development, production build, and GitHub Pages deployment
   commands in this file as the site is implemented.
